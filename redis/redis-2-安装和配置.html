# 安装和配置

# 安装

# docker安装

1、下载镜像

docker pull  redis

2、运行容器

docker run -d --name redis -p 6379:6379 \
  -v /data/docker/redis/data:/data \
  -v /data/docker/redis/conf/redis.conf:/etc/redis/redis.conf \
  --privileged=true redis:latest --appendonly yes --requirepass "123456"

注意：/etc/redis/redis.conf 是个目录

创建目录：

mkdir /data/docker/redis/data

mkdir /data/docker/redis/conf

创建 /data/docker/redis/conf/redis.conf

bind 0.0.0.0
protected-mode no
port 6379
tcp-backlog 511
timeout 0
tcp-keepalive 300

3、查看容器

docker ps

4、开发端口

firewall-cmd --zone=public --add-port=6379/tcp --permanent

5、进入redis-cli

docker exec -it redis redis-cli -p 6379 -a '123456'

# docker-compose安装

version: '3.1'
services:
  db:
    image: redis
    container_name: redis
    restart: always
    command: redis-server /etc/redis/redis.conf --appendonly yes --requirepass 123456
    ports:
      - 6379:6379
    volumes:
      - /data/docker/redis/data:/data
      - /data/docker/redis/conf/redis.conf:/etc/redis/redis.conf

# Redis 配置

Redis 的配置文件位于 Redis 安装目录下，文件名为 redis.conf。

你可以通过 CONFIG 命令查看或设置配置项：

127.0.0.1:6379> config get loglevel
1) "loglevel"
2) "notice"

使用 ***** 号获取所有配置项：

127.0.0.1:6379> CONFIG GET *
  1) "dbfilename"
  2) "dump.rdb"
  3) "requirepass"
  4) "123456"
  5) "masterauth"
  6) ""

可以通过修改 redis.conf 文件或使用 CONFIG set 命令来修改配置。

redis 127.0.0.1:6379> CONFIG SET loglevel "notice"
OK
redis 127.0.0.1:6379> CONFIG GET loglevel

1) "loglevel"
2) "notice"
